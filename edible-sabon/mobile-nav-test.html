Mobile Navbar Test

This page tests the responsive navbar with hamburger menu functionality.

Current URL: http://localhost:5174/entrep-website/

Testing Instructions:

  1. Use the buttons below to test different screen sizes
  2. Click the hamburger menu icon (☰) on mobile sizes
  3. Verify the menu slides in from the right
  4. Click menu items or the overlay to close
  5. Test the close icon (×) appears when menu is open

Note for Mobile Testing:

If testing on an actual mobile device, ensure you're on the same network and use the local IP address.

You can also use Chrome DevTools (F12) → Toggle Device Toolbar (Ctrl+Shift+M) to simulate mobile devices.
