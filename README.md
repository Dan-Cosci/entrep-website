# College Entrepreneurship Project

## Project Overview
This repository contains the initial development for our group project in the Entrepreneurship course. Our goal is to build a viable business solution that addresses a specific market need.

## Tech Stack (Frontend)
For the current phase of the project, we are focusing on the user interface and experience using the following technologies:

*   **Framework:** React.js / Next.js
*   **Styling:** Tailwind CSS
*   **Icons:** Lucide React

## Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
